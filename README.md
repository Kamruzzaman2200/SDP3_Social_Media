## üìñ About
**SDP3_Social_Media** is designed to provide a seamless social networking experience. The goal of this project was to build a scalable application that handles user authentication, data storage, and real-time interaction, demonstrating proficiency in full-stack development.

## ‚ú® Features

* **User Authentication**: Secure Login and Registration (Email/Password, Google Auth).
* **User Profiles**: customizable profiles with profile pictures, bios, and activity logs.
* **News Feed**: Dynamic scrollable feed showing posts from friends/followers.
* **Create Posts**: Upload text, images, and videos.
* **Interactions**: Like, Comment, and Share posts.
* **Real-time Chat**: Direct messaging between users.
* **Notifications**: Real-time alerts for likes, comments, and new messages.
* **Search**: Find users and posts easily.

## üöÄ Installation

Follow these steps to set up the project locally.

### Prerequisites
* [e.g., Node.js / Flutter SDK / JDK]
* [e.g., Android Studio / VS Code]

### Steps

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/Kamruzzaman2200/SDP3_Social_Media.git](https://github.com/Kamruzzaman2200/SDP3_Social_Media.git)
    cd SDP3_Social_Media
    ```

2.  **Install dependencies**
    ```bash
    # Example for Node/React
    npm install

    # Example for Flutter
    flutter pub get
    ```

3.  **Configure Environment Variables**
    * Create a `.env` file (or `config.js`) in the root directory.
    * Add your API keys (Firebase config, Database URL, etc.).

4.  **Run the App**
    ```bash
    # Example for Node/React
    npm start

    # Example for Flutter
    flutter run
    ```
